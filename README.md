# Full Stack Shopping System with Integrated Payment System


![Shopping System Banner](https://i.ibb.co/18PT8W6/Whats-App-Image-2024-06-02-at-12-49-44-bade02b8.jpg)

## Installation and Usage

To get started with the Shopping System, follow these steps:

1. Open the File, extract it to a particular folder and then open the folder in Visual Studio Code (Recommended) or any other editor.

2. After opening the folder in VS Code, open the terminal and navigate to both frontend and backend folder via running the command "cd/backend" and "cd/frontend" in split terminal mode of VS Code or by opening two seperate terminals.

3. Now Install the Dependencies by running the command "npm install" in both the terminals where you have navigated to frontend and backend folders.

4. After installing the dependencies, first run the backend server by running the command "npm run start-dev" and then start the frontend server by running the command "npm run dev".
